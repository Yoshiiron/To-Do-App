## How to run the App

### Local Run
```
git clone git@github.com:Yoshiiron/To-Do-App.git
cd backend go run .
cd ../frontend/src
npm start
```

### Docker
```
git clone git@github.com:Yoshiiron/To-Do-App.git
docker compose up
```
